# NEU Lab Room Monitoring System (CICS)

A real-time laboratory usage and monitoring system designed for the **College of Informatics and Computing Studies (CICS)** at New Era University. This system allows faculty members to log room usage, track program-specific analytics, and monitor room availability in real-time.

## 🚀 Live Demo
**Hosting URL:** [https://studio-9406975188-1a63d.web.app](https://studio-9406975188-1a63d.web.app)

## ✨ Features
* **Real-Time Dashboard:** Monitor 10 laboratory rooms (Lab 101 - 110) with live availability status.
* **Role-Based Access Control (RBAC):** Distinct permissions for Admin and Professor roles.
* **Administrative Security:** Capability to restrict user access (Read-Only mode) via the `isBlocked` status.
* **Program Analytics:** Track usage across BSIT, BSCS, BSIS, and BSEMC programs.
* **Automated Logging:** Single-record session management to prevent redundant data.

## 🛠️ Tech Stack
* **Framework:** Next.js 15
* **Database:** Firebase Firestore
* **Authentication:** Firebase Auth (Google Provider)
* **Hosting:** Firebase Hosting
* **Styling:** Tailwind CSS
